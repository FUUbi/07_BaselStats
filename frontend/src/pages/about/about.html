About BaselStats

Nowadays, an enormous amount of data is generated every day. Unfortunately, this data is mostly available on different sources and stored in different file formats.

BaselStat is an open data search engine that allows intuitive search for multivariant data available from the Kanton Basel Stadt.

It includes a using a public API that allows other developers to create their own applications based on the data. In addition to the Web Application, the project also includes a visualisation of the search results by using the Microsoft HoloLens.
