BaselStats

Statistic Data API


The public API provides access to the statistical data available in Canton Basel Stadt: http://baselhack.nicolasmauchle.ch


/stats

Returns all the available data sets.

Each data set entry is identified by a key that can be used for the data search.

Additional information is a title and a description of the data set.


/stats/<key>

Returns all the available data for the specified key, from all the available years.


/mapStats/<key>

Returns all the available data for the specified key, from all the available districts.


/auto?term=<term>

An API for auto complition of search terms. Returns all the available data sets with a description matching the requested term.

The input parameter term must contain at least 4 charachters.


/bezirke

Returns all the available districts, including their name and coordinates information.


/compare/<key>?from=<year>&to=<year>&with=<another_key>

Returns all the available data for the specified key.

Optional parameters are year from, year to and antother key of an additional data set for comparison.
